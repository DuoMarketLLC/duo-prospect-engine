# DUO Prospect Engine Spec

Goal:
Build a prospecting engine that finds, scores, and organizes leads for DUO services.

Services:
- eBroker
- BidCloser
- BidRescue
- Plug & Play Marketer
- Plug & Play Lite

Core Functions:
1. Find businesses by category
2. Capture business details (name, website, location)
3. Assign a service fit
4. Score the lead
5. Recommend the best DUO offer
6. Output clean data for CRM and outreach

Important Notes:
- I am not a programmer
- I want clean, simple, understandable code
- Build this in phases
- Start with a simple working version first

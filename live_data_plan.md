# Live Data Plan

## First live data source
Google Maps

## Why Google Maps first
- strong coverage for local businesses
- ideal for trades and estimate-based businesses
- useful for real estate and service businesses
- provides business name, category, address, phone, website, reviews

## Initial goal
Pull real business listings into the DUO Prospect Engine so leads can be scored using actual business data instead of mock data.

## First use cases
- trades_bidcloser
- real_estate_agents
- cpg_brokerage where applicable

## First live fields to capture
- business_name
- category
- website_url
- phone
- address
- city
- state
- rating
- review_count

## Rule
Keep the first live version simple.
Do not try to build everything at once.
Start with one search, one location, and a clean output.
